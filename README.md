SISANJAW (Sistem Informasi Pemesanan WarJAW) merupakan aplikasi berbasis web yang dikembangkan untuk mendukung proses pemesanan makanan di **WarJAW (Warung Jawa)**, yaitu kantin resmi Fakultas Teknik Universitas Sebelas Maret (FT UNS).

Sistem ini dirancang untuk meningkatkan efisiensi pelayanan, mengurangi antrean pemesanan, serta mendigitalisasi pencatatan transaksi secara terstruktur dan terintegrasi.

---

## ğŸ¯ Latar Belakang

WarJAW melayani mahasiswa dan civitas akademika Fakultas Teknik UNS setiap harinya. Proses pemesanan secara manual berpotensi menyebabkan:

- Antrean panjang saat jam sibuk
- Kesalahan pencatatan pesanan
- Rekap transaksi yang kurang optimal
- Kesulitan monitoring pesanan oleh pengelola

SISANJAW dikembangkan sebagai solusi digital berbasis Laravel untuk mengoptimalkan proses pemesanan dan manajemen transaksi secara real-time.
